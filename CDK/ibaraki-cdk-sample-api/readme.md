# AWS CDK

## install

install packages for lambda function
```
cd ibaraki-sample-lambda && npm i && cd -
```

install packages for cdk
```
npm i
```

## build
```
npm run build
```


## deploy
```
cdk deploy --profile {profile}
```

## delete
```
cdk destroy --profile {profile}
```
